// 03 — Security Overview

Capstone

HTTPS / TLS Encryption

All data in transit is encrypted. No plaintext. Cloudflare handles certificate issuance and renewal.

Static Hosting

No backend server means no server to exploit. GitHub Pages eliminates an entire attack surface category.

DDoS & DNS Protection

Cloudflare proxies all traffic, absorbs volumetric attacks, and provides secure DNS resolution.

Principle of Least Privilege

The site exposes only what it needs to. Contact email gated behind CAPTCHA to prevent scraping and spam.
